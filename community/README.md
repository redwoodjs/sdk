# rwsdk-community

Common extensions, utilities, and community-contributed features for [RedwoodSDK](https://www.npmjs.com/package/rwsdk).

> [!WARNING]
> **Stability Policy**: This package follows a separate stability policy from the core RedwoodSDK. Features in `rwsdk-community` may undergo breaking changes at any time without a major version bump. Use with caution in critical production environments.

## Contributing

Contributions are welcome! Please see the [Contributing Guide](https://github.com/redwoodjs/sdk/blob/main/CONTRIBUTING.md) in the main repository.
