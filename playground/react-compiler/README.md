# React Compiler

This playground example enables the React Compiler via Vite.

## Setup

```shell
npm install
```

## Running the dev server

```shell
npm run dev
```

Point your browser to the URL displayed in the terminal (e.g. `http://localhost:5173/`).

## Notes

- See `docs/src/content/docs/guides/optimize/react-compiler.mdx` for the setup steps this example follows.
