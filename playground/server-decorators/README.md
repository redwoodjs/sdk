# RedwoodSDK Minimal Starter

This starter gives you a bare-bones RedwoodSDK project which demonstrates the use of decorators in server-only files.

Create your new project:

```shell
npx create-rwsdk my-project-name
cd my-project-name
npm install
```

## Running the dev server

```shell
npm run dev
```

Point your browser to the URL displayed in the terminal (e.g. `http://localhost:5173/`). You should see the RedwoodSDK welcome page in your browser.
